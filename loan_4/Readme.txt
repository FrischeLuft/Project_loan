Резюме по слайдерам:

1) В проекте 5 слайдеров: 2 глобальных и 3 не глобальных;

2) Чтобы не создавать 5 различных модулей -> мы объеденили все в один класс;

3) Сущ. один  большой прототип, который называется Slider => в него мы можем передать параметры,
которые могут понадобиться этому слайдеру => все это передается в качестве объекта и применяется деструктуризация => тем самым мы можем вытаскивать определенные значения по надобности (page, btns и тд.);

4) Для того, чтобы мы могли создать один глобальный слайдер мы создали класс MainSlider и он будет наследоваться от Slider =>
в конструкторе говорим, какие свойства нам понадобятся из прототипа => далее исп. методы; 

5) Далее для создание глобального класса создаем new Main-Slider => передаем туда нужные значения;
6) Теперь у нас существует наследование классов;
